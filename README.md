## 🎮 주요 기능
- **실시간 멀티플레이어** 게임 지원  
- **역할 기반 마피아 게임** (마피아, 조커)  
- **유저 간 실시간 채팅**  
- 게임 진행에 따라 **자동 상태 전환**
